# Owner Action Checklist: SEO & Indexing

The following tasks require access to external accounts (Google, Bing, Domain Registrar).

## High Priority (Immediate)
*   [ ] **Verify GSC Domain Property**: Add `yagacalls.com` as a Domain Property in Google Search Console. Verify via DNS TXT record.
*   [ ] **Submit Sitemap (Google)**: Submit `https://www.yagacalls.com/sitemap.xml` to the GSC Sitemap report.
*   [ ] **Request Indexing (Priority URLs)**: Use the "URL Inspection" tool for the Homepage, About, and Review pages to request indexing.
*   [ ] **Verify Bing Webmaster Tools**: Import the site from GSC or verify manually. Submit the sitemap in Bing.

## Medium Priority (Weekly)
*   [ ] **Check Indexing Progress**: Review GSC every Tuesday/Friday to see which pages have moved from "Discovered" to "Indexed".
*   [ ] **Audit Query Performance**: Check which brand aliases (Yagacall, Yaga) are generating impressions.
*   [ ] **External Profile Verification**: Ensure all official profiles (Binance, Telegram) are cleaned as per the Phase 6 report.

## Technical Maintenance
*   [ ] **IndexNow Evaluation**: Consider adding the IndexNow plugin/code if Bing indexing is slow.
*   [ ] **Analytics Integration**: Confirm if event tracking (CTA clicks) is being recorded correctly.
