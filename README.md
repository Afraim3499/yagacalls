**Yaga Calls** is a premium, high-conversion crypto signals platform designed to bridge the gap between speculative trading and professional market analysis. Built with a focus on **Risk, Timing, and Market Narratives**, the platform provides a systematic approach to crypto trading through manual Telegram onboarding and research-backed signal delivery.

## 🚀 Core Features

- **Narrative Killer Method**: A proprietary signal generation strategy focusing on macro narratives and on-chain velocity.
- **Premium Pricing UX**: High-converting 3-tier pricing model with promotional discounting and manual verification.
- **Dynamic Academy & Blog**: SEO-optimized content modules designed for learner onboarding and conversion.
- **Interactive Scenario Calculator**: Real-time outcome modeling for risk-aware position sizing.
- **Trust-First Compliance**: Built-in risk disclosures and transparency sections to ensure institutional-grade clarity.
- **Modern UI/UX**: Dark-mode aesthetic with gold accents, glassmorphism effects, and fluid Framer Motion animations.

## ⚙️ Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Afraim3499/yagacalls.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
